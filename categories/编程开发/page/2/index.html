HTTP1.1状态码

       一、常用的状态码
200 OK，请求成功
301 Moved Permanently（永久的移除），请求的网页已永久移动到新位置
404 Not Found（未找到），服务器找不到请求的网页
500 Internal Server Error（内部服务器错误），服务器遇到错误，无法完成请求
503 Service Unavailable（服务不可用），服务器由于超载或维护目前暂时无法使用
       二、五大类状态码
   1）Informational 1xx
1开头的状态码表征一个临时响应，而且HTTP/1.0没有定义1xx状态码
   2）Successful 2xx
2开头的状态码表示客户端的请求成功接收
   3）Redirection（重定向） 3xx
3开头的状态码表示完成请求需要进一步的操作
   4）Client Error 4xx
4开头的状态码考虑客户端可能出错
   5）Server Error 5xx
5开头的状态码表示服务器知道出错了或不能完成请求

更详细的信息可查看：http://www.w3.org/Protocols/rfc2616/rfc2616-sec10.html
